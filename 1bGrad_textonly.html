$$ \Omega = \int \nabla \Omega \cdot \mathbf{ dl }$$

For a given vector field, if this vector field is conservative, it is the gradient function of some scalar function $\Omega$.

Visualisation Guide
  1. On the right, you can see a gradient field and its scalar parent function. Change the value of the parameter $a$, and observe the change in $\nabla \Omega$ and $\Omega$.
  2. Note that as the magnitude of $a$ increases, the length of the arrows in $\nabla \Omega$ increases and the contour of the parent function $\Omega$ becomes steeper as well. Hence, the gradient vector of $\Omega$ at any point increases as well.
  3. All gradient fields shown in this suite are conservative. This means that the line integral along the field, $\int \nabla \Omega \cdot \mathbf{ dl }$ is path independent.
